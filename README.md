## Setup

```
On Terminal:

1. cd server
2. npm i
3. npm run server
4. cd client
5. npm i
6. npm run dev

```

In the 'server' folder create an .env file and add these lines:

- OPENAI_API_KEY= "your OpenAI API key"
- PORT: "whatever port you want"
